Development Guide
#################

.. _development_index:

Welcome to the My_diffbot development documentation. This section provides detailed guides
for developers working on extending and customizing the My_diffbot robot.

.. note::
   This documentation is intended for developers who want to understand, modify, or
   extend the My_diffbot software stack. Familiarity with ROS 2 and ``ros2_control`` is recommended.

Available Guides

Hardware Interface Development
  Learn how to develop and customize the ``ros2_control`` hardware interface for
  My_diffbot's differential-drive system. This guide covers the architecture, implementation
  details, and best practices for working with the ``My_diffbotSystemHardware`` class.

  See :doc:`control` for more information.
